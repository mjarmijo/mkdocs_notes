ctrl + y  = open the quick explorer, 0 enter enter to be in the inbox
ctrl + m = move your note to somewhere
ctrl + n = new note (then move to inbox)

ctrl + tab = go forward a new tab
ctrl + shift + tab = go back a new tab

Vim shortcuts:
:3 = jump to line 3
3Y  = yank 3 lines inclusive
12dd = delete 12 lines inclusive
