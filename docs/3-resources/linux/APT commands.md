## Summary

`sudo apt update && sudo apt upgrade` - download list of available packages and upgrade installed ones to the latest version

`apt search htop` - search for packages 
`apt show htop` - show package information
`sudo apt install htop` - installs a package

`sudo apt remove htop` - remove package, but keep config files
`sudo apt purge htop` - remove package and config files

`apt list --installed` - show installed packages
`apt list --installed | grep htop` - filter installed packages
